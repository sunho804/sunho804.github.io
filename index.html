Neat Feat

ReactJS react-redux

A user-friendly application for machine learning engineer to review, manipulate, and clean their data without having any programming language or experience.

This application allows user to upload one or more csv file and show the raw data in tabular format, calculate the data and show meaningful numbers of the data in another table. The user can select rows to select, delete and select columns to transform to handle missing data, scale, normalize, mathematically combine, add polynomial features or delete the selected columns. They also can generate histogram and scatter chart with the data. After manipulation, the user can download the cleaned, transformed data as a csv file.

Pictaphone

Kotlin Android Studio Text Recognition Translation

A simple android translation application.

This application allows user to take a photo, use their own image, or simply type in their own text to translate to English. The app uses MLKit OCR to recognize the text within the image and MLKit Translator to detect the language of the text and translate. It also has Text to Speech functionality which allows for people with visual impairment to take a photo of a sign and translate it, and be able to hear it on their phone if the are not able to read the writing.
